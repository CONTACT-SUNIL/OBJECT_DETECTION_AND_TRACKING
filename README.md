# OBJECT_DETECTION_AND_TRACKING
# YOLOv3-Tiny Object Detection Project

## Description
This project performs real-time object detection using YOLOv3-Tiny and OpenCV.

## Files
- detect.py - Object detection program
- webcam.py - Real-time webcam detection
- coco.names - Object class labels
- yolov3-tiny.cfg - YOLO configuration file

## Model
The YOLOv3-Tiny weights file is required to run the project.

Download:
yolov3-tiny.weights

Place the weights file in the same folder as the Python files.

## Technologies Used
- Python
- OpenCV
- YOLOv3-Tiny
